DOM - Document Object Model

  1. O que é o DOM e qual é a sua importância na web?

    O DOM (Document Object Model) é uma interface de programação que representa documentos HTML e XML como uma árvore de objetos. Ele permite a manipulação da estrutura do documento, possibilitando alterações dinâmicas no conteúdo, estrutura e estilo da página, além de criar interatividade nas páginas web.

  2. Como o DOM relaciona a estrutura HTML de uma página com o navegador?

    O navegador converte o HTML em uma árvore de DOM durante o carregamento da página. Cada elemento HTML se torna um nó na árvore do DOM. O navegador usa essa árvore para renderizar a página, e alterações no DOM afetam a renderização.

  3. Como o DOM organiza os elementos de uma página web em uma estrutura hierárquica?

    O DOM organiza os elementos como uma árvore hierárquica, com diferentes tipos de nós, como:

    • Nó do Documento: Raiz da árvore.
    • Elementos: Representam tags HTML.
    • Atributos: Associados aos elementos.
    • Texto: Conteúdo textual dos elementos.

  4. Exemplos de como essa hierarquia é representada

    Aqui está um exemplo simples de como a hierarquia do DOM pode ser representada:

                    Document
                 └── html
                     ├── head
                     │   └── title
                     │       └── "Minha Página"
                     └── body
                         ├── h1
                         │   └── "Olá, Mundo!"
                         ├── p
                         │   └── "Bem-vindo à minha página."
                         └── a
                             ├── "Visite meu site"
                             └── href="https://exemplo.com"

  5. Como JavaScript pode ser usado para acessar, manipular e modificar elementos do DOM?

    JavaScript pode acessar, manipular e modificar elementos do DOM através de uma variedade de métodos e propriedades, como:

    • Selecionar Elementos: Usando métodos como document.getElementById() e document.querySelector().
    • Modificar Conteúdo: Alterar o texto ou HTML dentro de um elemento usando propriedades como innerText e innerHTML.
    • Alterar Estilos: Modificar o CSS de um elemento usando a propriedade style.
    • Adicionar/Remover Elementos: Usar métodos como appendChild(), removeChild() e createElement().

  6. Exemplos de métodos comuns do DOM

    • document.getElementById('id'): Seleciona um elemento com o ID especificado.
    • document.querySelector('.classe'): Seleciona o primeiro elemento que corresponde ao seletor CSS fornecido.
    • element.innerText: Altera o texto dentro de um elemento.
    • element.setAttribute('attribute', 'value'): Define um atributo para um elemento.
    • element.addEventListener('event', function): Adiciona um manipulador de eventos para o elemento.

  7. Como os eventos funcionam no DOM?

    Os eventos no DOM funcionam como sinais que informam quando algo acontece na página web, como cliques, digitação ou movimentação do mouse. Quando um evento ocorre, ele é capturado pelo DOM e pode ser tratado usando manipuladores de eventos.

    • Captura e Propagação: Os eventos podem ser capturados em diferentes fases: captura (de cima para baixo) e bolha (de baixo para cima).
    • Manipuladores de Eventos: Funções que são executadas em resposta a um evento específico.
